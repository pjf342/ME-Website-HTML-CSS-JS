Levitate Technologies, Inc - Senior Design

At Levitate Technologies, the task was to create a new mechanism for arm rotation that can easily be integrated into their current arm support system. The main goal for the design was to increase the cycles till failure by switching to a inflexible mechanism. This goal was successfuly met using non-circular gears and a spiral torsion spring.

The initial 2D design was developed in MATLAB. The data was exported into SOLIDWORKS to be converted into 3D. The gears were rapidly prototyped using FDM 3D printing. The mechanism was tested using a torque tester and the results were plotted using Google Sheets
