Levitate Technologies, Inc - Senior Design    ( Poster )

At Levitate Technologies, the task was to create a new mechanism for arm rotation that can easily be integrated into their current arm support system. The main goal for the design was to increase the cycles till failure by switching to a inflexible mechanism. This goal was successfuly met using non-circular gears and a spiral torsion spring.

The initial 2D design was developed in MATLAB. The data was exported into SOLIDWORKS to be converted into 3D. The gears were rapidly prototyped using FDM 3D printing. The mechanism was tested using a torque tester and the results were plotted using Google Sheets.

San Diego State University - Undergraduate Research Assistant

At San Diego State University, I was an undergraduate research assistant. The first research project was 3D printable biosensors. I designed various patterns to be FDM 3D printed using piezoelectric filament and performed basic static load analysis for displacement.

In another research project, I programmed an ESP32 microcontroller to wirelessly gather data from various sensors simultaneously via bluetooth. Additionally, I developed a Python program with a GUI to record and plot the data live, as well as preprocess the data for machine learning. The preprocessing included data normalization and statistical feature extraction.

In a continuation of the previous research project, I programmed ESP32 microcontroller to gather data from a 6-axis gyroscope+accelerometer sensor. I developed a Python program that recorded and plotted the data live. The program also allowed the user to obtain the data wired via serial communication, or wirelessly via a local Wi-Fi webserver established by the ESP32.
